ASCII Art player
================

This is a sketched project to demonstrate the FFMpeg API for playing videos.
It just consumes an arbitrary video file and plays it as fast as possible on the
terminal. 

Dependencies
------------

- ffmpeg (latest git version)
    * libavcodec
    * libavformat
    * libswscale

Build and play
--------------

    make
    ./asciiplayer <video>
